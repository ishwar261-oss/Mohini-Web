👋 What is MOHINI AI?

MOHINI AI is a friendly, intelligent assistant designed to help you learn concepts, solve problems, and get guidance using advanced AI. Think of it as a super-smart digital tutor that makes complex things easy.

🧩 What problems it solves?

Information online is often confusing and overwhelming. MOHINI cuts through the noise, breaking down hard topics into bite-sized, simple pieces. It fixes code, organizes tasks, and answers tough questions clearly.

💡 How it helps users?

It saves you time and frustration. Whether you are a student or a beginner learning to code, MOHINI creates visual diagrams, quick quizzes, and simple, step-by-step explanations to ensure you truly understand.

🔄 Code Flow

How MOHINI
Processes Your Input

From keystroke to AI response — watch the full data pipeline.

01
⌨️

User Input

User types a message or slash command in the ChatPanel. Input is captured by the Swing text field's action listener.

ChatPanel.java
sendButton.addActionListener(e -> {
  String input = inputField.getText().trim();
  if (!input.isEmpty()) processInput(input);
});
02
🔍

Validation & Routing

InputValidator sanitizes the input. CommandHandler checks if it starts with / and routes to the correct command class.

CommandHandler.java
if (input.startsWith("/")) {
  Command cmd = registry.get(input.split(" ")[0]);
  return cmd != null ? cmd.execute(args) : "Unknown command";
} else {
  return aiService.chat(input); // → AI path
}
Branching Logic
🤖

AI Chat Path

OpenAIService.java builds JSON with message history + memory context, calls OpenRouter API.

POST api.openrouter.ai/api/v1/chat
// model: openai/gpt-4o-mini
// messages: [{role, content}...]
// + MemoryManager context
OR
⚡

Command Path

Matched command runs locally: /calc, /quiz, /learn, /health, or data persistence commands. 

class CalcCommand implements Command {
  public String execute(String args) {
    return Evaluator.eval(args);
  }
}
04
💾

Storage Layer

Data persists via TodoManager, NotesManager, ProgressTracker, MemoryManager as JSON files in data/.

MemoryManager.java
public void remember(String key, String value) {
  memories.put(key, value);
  saveMemory(); // → data/memory.json
}

🎨

UI Rendering

Response rendered in the appropriate panel: ChatPanel, QuizPanel, VisualizationPanel, HealthPanel, or ProgressDashboard. Optional TTS output.

private void switchTab(String tabId) {
  cardLayout.show(contentPanel, tabId);
